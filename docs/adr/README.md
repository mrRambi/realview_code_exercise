# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Book Author Search App.

An ADR documents a significant architectural decision: the context that led to it, the decision itself, and its consequences.

## Format

Each ADR follows this structure:
- **Status** — Proposed / Accepted / Deprecated / Superseded
- **Context** — Why was this decision needed?
- **Decision** — What was decided?
- **Consequences** — What are the trade-offs?
